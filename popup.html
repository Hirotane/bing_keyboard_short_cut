Keyboad Shortcut

If you want to enable/disable, please go to option page.

Navigation on search results
j/↓: move to next search result
k/↑: move to previous search result
l/→: move to next search result page
h/←: move to previous search result page
/: focus on search box
ESC/Ctrl+[: unfocus on search box
Enter: open selected result
Shift-a: search from all
Shift-w: search from work
Shift-i: search from image
Shift-n: search from news
Shift-v: search from video
Shift-m: search from map
Shift-s: search from shopping
Ctrl-g: search on google
Ctrl-b: search on bing
Ctrl-e: show search result in english
Ctrl-d: show search result in default language
Page transition
Shift-l: go back
Shift-h: go forward
Scroll operation in web sites
j/↓: scroll down
k/↑: scroll up
Ctrl-d: scroll down (Large)
Ctrl-u: scroll up (Large)
