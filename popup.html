Keyboad Shortcut

If you want to enable/disable, please go to option page.

Navigation on search results
j/↓: move to next search result
k/↑: move to previous search result
l/→: move to next search result page
h/←: move to previous search result page
/: focus on search box
ESC/Ctrl-[: unfocus on search box
Enter: open selected result
a: go to all tab
w: go to work tab
i: go to image tab
n: go to news tab
v: go to video tab
m: go to map tab
s: go to shopping tab
Page transition
Shift-h: go back
Shift-l: go forward
Switch search engine
Ctrl-g: search on Google
Ctrl-b: search on Bing
Change language
Ctrl-e: show search results in english
Ctrl-d: show search results in default language
Scroll operations in all web sites
j/↓: scroll down
k/↑: scroll up
d: scroll down (Large)
u: scroll up (Large)
gg: scroll to top
Shift-g: scroll to bottom
